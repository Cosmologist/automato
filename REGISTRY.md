# Registry

- `amneziawg/client` — List AmneziaWG client profiles
- `system/apt/repository` — List APT repositories and fetch remote components
- `system/network/interface` — Show network interface info
- `system/software/manager` — Universal way to execute or install software from various sources (currently supports apt and GitHub)
- `app/mybot` — Minimal Telegram bot that replies to direct messages and group mentions
